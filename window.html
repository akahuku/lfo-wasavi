The local directory to export to Chrome browser:

N/A
Please select the directory to export to Chrome. This directory becomes the root in the application. Basically, this should be assigned your home directory itself or any subfolder within your home directory.

The absolute path to your home directory

Please input the absolute path to your home directory. It is a string like:
  • Windows: C:\Users\your-account\
  • Linux: /home/your-account/
  • macOS: /Users/your-account/
